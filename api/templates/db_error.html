Database connection failed

{{ message }}

Fix

  1. Open Supabase → Project Settings → Database → Connection string.
  2. Copy the Session pooler URI (not a guessed aws-0-… host).
  3. Set DATABASE_URL in Vercel → Environment Variables (and local .env).
  4. Do not set CALLIT_USE_SQLITE=1 on Vercel.
  5. Redeploy, then check /health.
